# MediBot - AI Medical Health Chatbot

**MediBot** is an AI-powered medical information chatbot that provides **general health guidance** and **emergency alerts**. It uses **OpenAI embeddings** for semantic search through a medical PDF database and **Chainlit** for an interactive web interface.

⚠️ **Disclaimer:** MediBot is **not a substitute for a doctor**. Always consult a healthcare professional for serious symptoms or emergencies.

---

## Project Summary

MediBot is designed to:
- Understand user medical queries using **semantic search**.
- Provide answers using **GPT-4.1-mini** based on verified medical content.
- Detect **emergency keywords** (e.g., chest pain, stroke) and alert the user.
- Maintain **chat history** and allow **file uploads**.
- Offer a **modern, dark-themed UI** with a medical background and avatar.

This project is ideal for educational purposes, health information systems, and AI-powered personal assistants.

---

## Features

- ✅ Ask medical questions in natural language.
- ✅ Fast semantic search via **FAISS** vector database.
- ✅ Emergency symptom detection with immediate alerts.
- ✅ Upload files (PDFs, images, etc.) for context.
- ✅ Customizable dark-themed UI with medical background.
- ✅ Editable messages and chat session management.
- ✅ Chain of Thought (CoT) explanation enabled for responses.
